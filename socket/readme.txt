basic 基本收发方程序

1、rcver接收方：

 建立socket();
 端口和地址绑定bind();
 从网络上接收recvfrom();
 关闭网络；


2、snder发送方：
  
  建立socket();
  可以省略端口和地址绑定
  发送数据sendto();
  关闭网络；

3、proto协议：

  定义了接收方端口号
  定义了消息中名字字段的大小
  定义了发送接收消息的数据结构(采用不对齐格式)

4、涉及到的知识点：
   htons()：用来将主机上的值和用于网络传输值的转换
   inet_pton():将字符串形式的网络格式转换为大整数到网络上进行传输
   inet_ntop():将IPV4地址由大整数转化为字符串


 ******************************************************
 var ：basic基础上的变长name名字字段
      通过更改结构体中name长度，将其置1
	  设置最长报文内容长度
	  在rcver和snder方分别malloc一个最大长度空间，用来存放从命令行输入的name      字段

 bcast:basic基础上的广播实现

 远端地址：255.255.255.255
 加入一个广播设置函数setsockopt();
